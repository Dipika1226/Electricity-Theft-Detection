# Dataset Information

This folder contains the dataset used for the project **Electricity Theft Detection using Machine Learning**.

## File

- `data.zip`

> The CSV file was compressed to a ZIP format due to GitHub file size limitations.

## Source

Dataset downloaded from Kaggle:  
https://www.kaggle.com/datasets/sreen28g10/electricity-theft-detection

## Description

- ~42,000 customer rows
- ~1000 daily electricity usage columns
- `FLAG` column indicates theft (1) vs normal (0)

## Notes

- Extract the ZIP before running the notebook.
- Place the dataset in the same folder structure for reproducibility.
